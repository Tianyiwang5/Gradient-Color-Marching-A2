## General Information
gradient color marching is a short stickmen animation that intends to address my own understanding of the
interactions between different people who have various
standpoints in the real -world


## Technologies Used
Processing 4.4.10

## Features
-stickmen of different colors have their own movements states and movement paths 

-a certain colorful stickman gradually switched its color to another color



## Usage
Once the file is open, click the "play/run" button located at the left top side of the file and the animation will begin to play.
